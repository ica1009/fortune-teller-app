# 前端：登录/注册与占卜主界面测试

## 环境

- 前端：http://localhost:10075（Docker）或 http://localhost:5174（本地 dev）
- 后端需已启动。

## 路由与导航（SPA）

- 路径：`/` 占卜主界面，`/login` 登录，`/register` 注册。
- 未登录访问 `/` 会重定向到 `/login`；已登录访问 `/login`、`/register` 会重定向到 `/`。
- 页面切换为前端路由，无整页刷新；有约 250ms 淡入+短距滑动过渡。

## 用例 F1：未登录显示登录页

| 项目 | 说明 |
|------|------|
| 步骤 | 清除 localStorage 或无痕窗口，打开前端首页。 |
| 预期 | 显示「🔮 高级算命」与登录表单。 |
| 需截图 | 是：登录页首屏。 |
| 通过标准 | 仅有登录表单，无占卜界面。 |

## 用例 F2：注册后跳转登录

| 项目 | 说明 |
|------|------|
| 步骤 | 点击「注册」，输入用户名（3–32 位）、密码（≥6 位），提交。 |
| 预期 | 注册成功后回到登录页，可用该账号登录。 |
| 需截图 | 是：注册页；注册成功后登录页。 |
| 通过标准 | 注册成功且能登录。 |

## 用例 F3：登录成功进入占卜界面

| 项目 | 说明 |
|------|------|
| 步骤 | 输入正确用户名与密码，点击登录。 |
| 预期 | 进入占卜主界面，顶部显示当前用户名与「退出」按钮。 |
| 需截图 | 是：登录后主界面（含用户名与退出）。 |
| 通过标准 | 可抽签占卜，点击退出回到登录页。 |

## 用例 F4：丝滑导航（无整页刷新）

| 项目 | 说明 |
|------|------|
| 步骤 | 在登录页点击「注册」；在注册页点击「去登录」；登录后点击「退出」。 |
| 预期 | 每次切换均为前端路由，无整页刷新；有淡入+短距上滑过渡；地址栏为 `/login`、`/register`、`/`。 |
| 需截图 | 可选：过渡中的某一帧或地址栏变化。 |
| 通过标准 | 无白屏整页重载；浏览器前进/后退可沿历史在登录/注册/主界面间切换；直接打开 `/` 未登录会跳到 `/login`。 |
